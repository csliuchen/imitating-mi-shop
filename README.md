# 用 Vue 高仿 PC 端小米商城 

## ✨前言
这是一个学习 Vue.js 过程中做的小米商城 PC 端网页
使用了包括但不限于以下插件和 Vue 组件
```
Vue
Vue-CLI
Vue-Router
Vuex
axios
sass
swiper
vue-lazyload
element-ui
```
